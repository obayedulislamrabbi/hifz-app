✅ আমরা কোনো ব্যক্তিগত তথ্য সংগ্রহ করি না।

এই অ্যাপটি সম্পূর্ণ অফলাইনে কাজ করতে পারে। আপনার সকল ডেটা (হিফজের অগ্রগতি, আমল, নোট, প্রিয় আয়াত) শুধুমাত্র আপনার ডিভাইসে সংরক্ষিত থাকে।

  • ❌ কোনো সার্ভারে ডেটা পাঠানো হয় না
  • ❌ কোনো ব্যক্তিগত তথ্য সংগ্রহ করা হয় না
  • ❌ কোনো তৃতীয় পক্ষের সাথে তথ্য শেয়ার করা হয় না
  • ✅ সকল ডেটা শুধু আপনার ডিভাইসে (localStorage)

2. ইন্টারনেট ব্যবহার (Internet Usage)

অ্যাপটি নিচের কারণে ইন্টারনেট ব্যবহার করে:

  • 📖 কুরআনের আয়াত লোড: alquran.cloud API (শুধু পড়ার জন্য)
  • 🕌 নামাজের সময়: aladhan.com API (অবস্থান ভিত্তিক)
  • 🎵 তেলাওয়াত অডিও: cdn.alquran.cloud (শুধু স্ট্রিমিং)
  • 🌤️ আবহাওয়া: open-meteo.com (background পরিবর্তনের জন্য)
  • 🤲 AI দোয়া: Anthropic API (শুধু আপনার অনুরোধে)

এই API গুলো থেকে কোনো ব্যক্তিগত তথ্য পাঠানো হয় না।
